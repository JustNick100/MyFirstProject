The Project Description:

I made 2 user types: admin and customer

Users, who has been registered as admin, are available to add items to web pages and also remove them.

And users, who has been registered as customers, are available to only view the items.

If a new user has been registered, to register him as either admin or customer, it is necessary to open admin panel.
